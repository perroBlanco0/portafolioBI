De un respaldo SQL Server en la nube a un Dashboard en Power BI

## 🚀 Descripción del Proyecto
Diseño y despliegue de un flujo completo de datos (Data Pipeline) en la nube, que abarca desde el aprovisionamiento de infraestructura cloud y la migración de datos de un respaldo físico, hasta el modelado relacional y la entrega de un dashboard interactivo de inteligencia de negocios.
